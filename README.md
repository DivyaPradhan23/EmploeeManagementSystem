# Employee Management System

## Description
This is a ColdFusion-based Employee Management System developed as part of an assignment.

## Features
- Add Employee
- Edit Employee
- Delete Employee (with confirmation)
- Email uniqueness validation (Create & Update)
- Search by Name, Email, or Mobile No (AJAX-based)
- Validation for empty search input
- Column sorting (ASC/DESC using AJAX)
- Alphabet filter (Aâ€“Z dynamic enable/disable)
- Export to Excel with formatted headers
- Table refresh without full page reload

## Technologies Used
- ColdFusion
- SQL Server
- jQuery
- Bootstrap

## Database Setup
1. Open SQL Server Management Studio
2. Create a new database
3. Run the provided SQL script (EmployeeManagement.sql)
4. Configure the datasource in Application.cfc

## How to Run
1. Place project folder inside ColdFusion web root
2. Configure datasource
3. Open employee.cfm in browser
